🚀 אפליקציית ווב Docker

ברוכים הבאים לאפליקציית הווב שלכם במיכל Docker!

זוהי אפליקציית ווב פשוטה שרצה בתוך מיכל Docker באמצעות Nginx.

מה שאתם יכולים לעשות בהמשך:

  • עריכת קובץ ה-HTML הזה ובנייה מחדש של המיכל
  • הוספת קבצים סטטיים נוספים לשרת הווב
  • יצירת Dockerfile רב-שלבי עבור אפליקציה מורכבת יותר
  • שימוש ב-Docker Compose לאורכיסטרציה של שירותים מרובים

מידע על המיכל:

תמונת בסיס: nginx:alpine

פורט: 80

סטטוס: פועל

פקודות Docker שימושיות:

# בניית התמונה
docker build -t my-webapp .

# הפעלת המיכל
docker run -d -p 8080:80 my-webapp

# צפייה במיכלים פועלים
docker ps

# עצירת המיכל
docker stop <container_id>
